## Descripcion
Sistema de gestión académica hecho en Java. El programa funciona en consola mediante un menu interactivo, y permite gestionar estudiantes, profesores y materias. El programa permite registrar registrar profesores y estudiantes, registrar calificaciones y realizar reportes de promedio, tambien puedes buscar estudiantes por matricula y nombre. Puedes usar el programa en cualquier computadora con JVM instalado.

## Tecnologías utilizadas
- Java
- JDK 25
- Visual Studio Code
